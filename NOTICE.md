# Avisos legais

O código-fonte do DualSenseHUB é disponibilizado sob a licença MIT.

O arquivo `assets/DualSenseHUB.ico` foi fornecido para ser usado como identidade
visual do projeto. Os demais recursos visuais de `assets/` são distribuídos para
permitir auditoria e reprodução do executável.

DualSense, PlayStation e PS5 são marcas de seus respectivos titulares. Este é um
projeto comunitário independente, sem vínculo, patrocínio ou aprovação da Sony
Interactive Entertainment.
